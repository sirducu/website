URL Shortener API

Example: python!
import requests
url = "http://radu.ovh/api/short"
querystring = {"URL" : "https://www.youtube.com/watch?v=AG56t8t9iW8"}
response = requests.get(url, params=querystring)
rsp = response.text
print(rsp["ShortURL"])
